# 项目路线图 (Project Roadmap)

## Phase 1: 基础设施与骨架搭建 (当前阶段)
- [x] 初始化前端项目 (Vue 3 + Vite + TS + Tailwind)。
- [x] 初始化后端项目 (Spring Boot 3 + Maven + MySQL + Redis 依赖配置)。
- [x] 数据库 Schema 初始化与实体类 (Entity) 代码生成。

## Phase 2: UI/UX 布局与像素级还原 (Updated)
- [x] Tailwind 配置文件精准映射 (写入 `#f4f5f7`, `16px` 圆角等自定义变量)。
- [x] 搭建精确的左右分栏骨架 (左侧 240px 固定白底侧边栏，右侧 Flex 伸缩区域包含 Header 和灰底主内容区)。
- [x] 细化顶部 Header 与分类栏 (包含高保真 Pill 形态搜索框与黑白对比的分类 Tag 列)。
- [x] 构建 Banner 区 (左侧 65% 大幅面主图，右侧 35% 浅绿渐变底色的“探索更多” 2x2 网格)。
- [x] 构建高保真模型卡片 `ModelCard` (无边框，16px圆角，底部悬浮信息栏，精细的图标与字体排版)。

## Phase 3: 后端核心业务 API 开发
- [x] 用户模块：Spring Security + JWT 鉴权配置。
- [x] 安全模块：基于 Redis + 自定义拦截器实现防撞库（登录限流策略）。
- [x] 存储模块：集成 OSS SDK，实现图片上传并返回 URL 的接口。
- [x] 模型模块：实现模型数据的发布 (POST) 接口。
- [x] 查询模块：基于 MyBatis-Plus 编写多条件动态 `LIKE` 查询及分页接口，并加入 Redis 缓存。

## Phase 4: 前端页面与联调
- [x] 封装 Axios 请求及全局响应/异常拦截器。
- [x] 编写登录与注册页面（集成前端行为验证码）。
- [x] 编写模型上传表单页面。
- [x] 编写模型大厅页面（支持瀑布流、分类 Tab、搜索框模糊检索）。

## Phase 5: 测试与部署
- [ ] 后端 Jmeter 并发查询压测及限流阈值调优。
- [x] Docker 化部署打包 (Dockerfile & docker-compose)。

## Phase 6: 基础体验增强 (当前阶段)
- [x] **数据结构升级**：执行 SQL 脚本，扩展 `tb_model` 打印参数字段。
- [x] **上传流重构**：在 `UploadView.vue` 增加打印设置表单与协议选择器。
- [x] **信息展示优化**：在详情页增加“打印建议”面板，展示层高、填充等专业信息。

## Phase 7: 社区互动功能实现
- [x] **作品秀 (Makes)**：开发后端上传接口，前端在模型下方实现“打印实物墙”。
- [x] **收藏夹系统**：实现简单的收藏逻辑，并在侧边栏/个人中心增加“我的收藏”入口。
- [x] **版权合规**：在模型卡片和详情页显著位置展示 CC 协议图标。

## Phase 8: 安全与性能优化
- [x] 接入 Cloudflare Turnstile 行为验证。
- [x] 图片懒加载与 WebP 格式转换优化。

## Phase 9: 平台后台治理能力（用户 / 角色 / 组织）
- [x] 明确平台级权限模型：菜单权限 + 接口权限 + 数据范围权限。
- [x] 建立树形组织管理能力：组织树查询、组织节点增删改、组织状态管理。
- [x] 建立用户管理能力：用户列表检索、创建编辑、启停用、重置密码、组织归属管理。
- [x] 建立角色管理能力：角色创建编辑、角色状态管理、多角色分配。
- [x] 建立授权能力：角色-菜单、角色-接口、角色-数据范围配置。
- [x] 完成权限校验链路改造：登录态、权限并集计算、数据范围过滤。
- [x] 生成后台基础种子数据 SQL：超级管理员账号、基础角色与默认授权关系。
- [ ] 建立后台审计能力：登录日志、操作日志、越权访问记录。
- [ ] 完成前后台联调与验收：管理页面可用、接口鉴权正确、数据范围生效。
